# Scripts

In diesem Ordner liegen alle **Hilfs- und Wartungsskripte**, die das Projekt unterstützen – z. B. Setup-, Deployment-, Datenaufbereitungs- oder Analyse-Skripte.

Diese Skripte dienen dazu, **Entwicklung, Test und Betrieb** des Systems effizienter zu gestalten.  
Sie sind **nicht Teil der Hauptlogik**, sondern unterstützen Aufgaben rund um Installation, Datenhandling, Simulation und Wartung.
